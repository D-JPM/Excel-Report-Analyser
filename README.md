# Report Analyser

**Description**
This is a small Python Program to process and analyse Excel Reports,
it is dependant on a specific but simple format.


**Features**
- Reads and processes the Excel file (Report).
- Peforms basic analytics that would otherwise be done manually (e.g. sums, percentages)
- Displays the results inside a GUI using Tkinter.

**How to Run**
1. Clone Repo
2. RUN 'gui.py'

**Dependencies**
- Python 3.10.12
- openpyxl
- pandas
- customtkinter
- tkinter(this should already be bundled in Python3 but wanted to include it in dependencies)
